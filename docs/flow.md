# Luồng Nghiệp Vụ

## Luồng Đơn Hàng
1. Người dùng thêm sản phẩm vào giỏ hàng
2. Người dùng thực hiện thanh toán
3. Hệ thống:
   - kiểm tra dữ liệu đầu vào
   - kiểm tra tồn kho
   - tạo đơn hàng
   - tạo order_details
   - trừ tồn kho
4. Trả về kết quả thành công

- Danh sách -> lọc -> phân trang
